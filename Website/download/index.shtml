Version 1.2.0 is another major step forward. I've introduced new features, new and updated components, and generally made all-around enhancements to the entire project.

I've added many components and made updates to information for them and using them with Modevious. We've got a pretty large set of tools from which we can make wonderful websites and applications out of!

  • Modevious Update System allows you to check for updates to any plugins that make use of the system!
  • jQuery UI updated to version 1.8.5
  • Updated our custom build of jQuery Tools
  • Added SoundManager 2
  • For users: Modular plugin-type packaging to make deployment even easier!
  • For developers: Re-organization of project and use of scripts that help automate things!

All updates after version 1.2.0 are cumulative. If you already installed and are using Modevious, simply grab the latest update you want and you've got all previous updates as well! Don't go backwards though... even though I cannot see any reason why you'd do so I'm willing to bet some of you will try or at least accidentally do it.
