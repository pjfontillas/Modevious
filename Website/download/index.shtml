This version of Modevious is a major step towards the next big release. This current version (v.1.0.9) covers:

  • A major re-write of the code for the Core. The Core now supports public and private members and methods.
  • Minor bug fixes for some functions and components.
  • Added project to GitHub to share with and invite others to participate. Also tracks bugs and issues for Modevious

Version 1.1.0 will cover:

  • jQuery UI updated to 1.8.2.
  • Bug reports for some components issued.
  • Setting up PivotalTracker for developers to collaborate on implementing new features and cleaning up code.
  • New components and added functionality.

All updates after version 1.0.9 are cumulative. If you already installed and are using Modevious, simply grab the latest update you want and you've got all previous updates as well! Don't go backwards though... even though I cannot see any reason why you'd do so I'm willing to bet some of you will try or at least accidentally do it.

Still on my To-Do List is a better update system. In a future release I am going to include a dashboard type of page that initializes a variable that activates the system. My hopes are that authors will use the system to allow developers and site maintainers to keep updated or at least informed of new releases by accessing the dashboard.

I also wish to include a demo or test page which you can use to experiment with.
