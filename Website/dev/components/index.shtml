autoMouseOver

autoMouseOver is a jQuery plugin that automatically adds a mouseover effect to any selected images based on their naming structure.

With the default settings, any selected image that has "-out." in its src attribute (eg. home-out.png) will try to mouseover to the same image name but with "-over." instead (eg. home-over.png). The default settings can be changed when the plugin is called.

Core

Library built specifically for Modevious. The Core library holds some commonly used functions. A couple can be found in other libraries, while others are new ideas unique to Modevious.

The Core library has dependencies on both the Prototype JS and jQuery libraries, utilizing tools from both libraries to create new ones. It also borrows concepts and techniques from various other authors on the web.

dumbCrossFade

jquery.dumbCrossFade is intended to be a light-weight slide transition that enables external manipulation. Tested to work with jQuery 1.4 and used in over 200 ecommerce websites as a base tool.

jQuery

jQuery is a new kind of JavaScript Library.

jQuery is a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. jQuery is designed to change the way that you write JavaScript.

jQuery UI

jQuery UI provides abstractions for low-level interaction and animation, advanced effects and high-level, themeable widgets, built on top of the jQuery JavaScript Library, that you can use to build highly interactive web applications.

MUpS

The Modevious Update System (MUpS) is an AJAX-based system designed to pull update data for components that register themselves with the system through the use of JSON formatted data. Bypasses the Same Origin Policy problems by cleaning data and limiting what third parties have access to.

It's like having all your feeds aggregated in one place and stripped down to only what you are looking for.

Pines Notify

The Pines Notify is a jQuery plugin developed by Hunter Perrin as part of Pines.

It is a notification system that uses the jQuery UI CSS library for styling, which means it is fully and easily themeable.

Prototype

Prototype is a JavaScript Framework that aims to ease development of dynamic web applications.

Featuring a unique, easy-to-use toolkit for class-driven development and the nicest Ajax library around, Prototype is quickly becoming the codebase of choice for web application developers everywhere.
