Core

Library built specifically for Modevious. The Core library holds some of the most common ajax functions used. Some can be found in other libraries, some are new ideas unique to Modevious.

The Core library has dependencies on both the Prototype JS and jQuery libraries, utilizing tools from both libraries to create new ones. It also borrows concepts and techniques from various other authors on the web.

Prototype

Prototype is a JavaScript Framework that aims to ease development of dynamic web applications.

Featuring a unique, easy-to-use toolkit for class-driven development and the nicest Ajax library around, Prototype is quickly becoming the codebase of choice for web application developers everywhere.

jQuery

jQuery is a new kind of JavaScript Library.

jQuery is a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. jQuery is designed to change the way that you write JavaScript.
