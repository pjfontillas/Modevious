"Merging Our DEVelopment Into One Universal System"

Modevious is a collection, testing tool, and maintenance system of JavaScript libraries and plugins.

Saying there will be one universal system may be over-reaching a bit, but the idea is that everything can work together.

Many tools essential to building an interactive site or application are already included in our library. No need to search for them.

Modevious also employs an update system that makes it easier for plugin authors and users to keep things up-to-date.

What is cloud computing?

According to Wikipedia:

Cloud computing is Internet-based computing, whereby shared resources, software, and information are provided to computers and other devices on demand, like the electricity grid.

How does that affect me?

With the agile progression of technology we are now at a point where just about any device with a web browser is capable of running JavaScript.

Rather than placing almost all of the work on the web server we are now able to leverage the power of individual computers to help process our web pages and applications.

Modevious attempts to alleviate the process of creating and maintaining web pages and applications that use client-based technologies to enhance our experience on the web.

Possible jQuery conlicts

If you already run jQuery scripts check to see if the author implemented the referencing method for running jQuery code.

One of the biggest problems with using the Prototype JS and jQuery libraries together was the conflicting use of '$'.

If the author did not protect calls to their script from this conflict you will have to convert their run-time script - the script or code that initializes their application. Change all references to '$' to '$j' to have it work with Modevious.

Mobile optimization

Modevious is designed for the desktop user experience. If you're looking for a library that is designed from the ground up to cater to mobile needs check out jQuery Mobile.

You can use Modevious on the mobile platform, you may just get some uncommon behavior or decreased performance based on the hardware it runs on.

jQuery UI

The jQuery UI is currently just about the extent of theming Modevious directly deals with.

Modevious comes with its own custom jQuery UI theme but feel free to roll your own. Just remember to replace the appropriate files and try to stay within the correct jQuery and jQuery UI versions.

Core CSS

Remember inline-styling? Don't use that.

For the most common applications of inline-styling there are Core CSS classes that will provide all the markup needed. Need to make the text in a certain paragraph left-aligned? Use the class "text-left". Need an element to float? Apply the class "float-left" or "float-right".

I encourage this use because that certain style may apply to only one page or a specific element, in which case I then believe it is better to leave it out of the global CSS rules.

Packaged Tools

Commonly available scripts and plugins have been pre-packaged to ease the burden of finding and collecting tools. They've also been tested to work together and shouldn't cause conflicts.

Configurable

Most aspects of Modevious can be tweaked to suit your needs. The default jQuery UI theme can be replaced, and several options can be set to favor your type of development environment. Or you can just turn the key and start using it right away without modifying anything!

Update System

Modevious comes with a system to easily keep your plugins updated. If you're an end-user you'll know which plugins are starting to get outdated, and as a developer you've got another outlet that you can use to inform your users of updates.

Console Available

Cross-browser console that can be logged to and has some basic functionality that can help you debug at a level built-in browser development tools sometimes fail to do.

Well Documented

If it's not covered on this website, the components page can point you to the right place to find the answers you're looking for.

Easy to use
